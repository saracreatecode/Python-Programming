# 🐍 Programación en Python — DAM

Repositorio dedicado al aprendizaje y práctica de **Python** dentro del ciclo formativo de **Desarrollo de Aplicaciones Multiplataforma (DAM)**.  
Aquí recopilo los **ejercicios, prácticas y proyectos** realizados durante el estudio de la programación con Python, aplicando tanto programación estructurada como conceptos de programación orientada a objetos.

---

## 🎯 Objetivos del módulo

- Comprender los **fundamentos de Python** y la lógica de programación.  
- Aprender a desarrollar programas utilizando **estructuras de control, funciones y módulos**.  
- Aplicar los principios de la **programación orientada a objetos (POO)** en Python.  
- Crear código **limpio, reutilizable y mantenible** siguiendo buenas prácticas.  
- Trabajar con **ficheros, colecciones, excepciones y librerías estándar** de Python.  

---

## 🧠 Contenidos

| Unidad | Tema | Descripción |
|:------:|------|-------------|
| 1 | Introducción a Python | Sintaxis básica, variables, tipos de datos y operadores |
| 2 | Estructuras de control | Condicionales, bucles y control de flujo |
| 3 | Funciones y módulos | Organización del código y reutilización |
| 4 | Programación orientada a objetos | Clases, objetos, atributos y métodos |
| 5 | Colecciones y estructuras de datos | Listas, tuplas, diccionarios y conjuntos |
| 6 | Gestión de ficheros | Lectura, escritura y manipulación de archivos |
| 7 | Manejo de excepciones | Control y gestión de errores en programas |
